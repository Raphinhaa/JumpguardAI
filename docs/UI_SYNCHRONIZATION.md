# UI Synchronization

Prompt 21 synchronization is implemented in the generated standalone interactive viewer. The synchronized state is the selected processed-frame index.

## Data Source

The viewer embeds the existing frame database generated by the Prompt 20 interactive analysis layer. Each frame record contains:

- `frame_index`,
- `timestamp`,
- per-frame joint-angle measurements,
- delta values from the previous processed frame,
- symmetry metrics,
- landmark confidence summaries, and
- landmark rows.

No additional numerical calculations are introduced by Prompt 21 beyond rendering already exported values in the UI.

## Synchronized Elements

The selected frame updates:

- frame number,
- timestamp,
- overlay labels,
- Frame Information cards,
- Joint Angles table,
- Delta Values table,
- Symmetry table,
- Landmark Confidence cards,
- knee, hip, ankle, delta, and symmetry graphs, and
- hip, knee, and ankle left/right comparison bars.

## Graph Cursor

Every graph draws a vertical red cursor at the selected processed-frame index. Clicking a graph pauses playback and selects the nearest processed frame on the graph.

## No Interpolation Policy

Displayed measurements correspond to the selected processed frame. The viewer does not interpolate, average, infer, or smooth values for display.
