# Clinician Workflow

The Prompt 21 workstation is designed for clinician-led biomechanical review.

## Workflow

1. Upload a jump-landing video.
2. JumpGuard runs the existing video, pose, annotation, and measurement pipeline.
3. Open the Professional Workstation output.
4. Review the annotated video.
5. Pause, scrub, step frame-by-frame, or jump to a timestamp.
6. Inspect measurements for the currently displayed frame.
7. Use synchronized graphs and comparison bars to understand the selected moment.
8. Exported CSV/JSON files remain available for independent review.

## Clinician Responsibility

The clinician selects the frame or moment of interest. JumpGuard displays measured values for that selected frame only.

## Explicit Non-Goals

Prompt 21 does not add:

- automatic event detection,
- initial-contact detection,
- peak-flexion detection,
- ACL risk prediction,
- diagnosis,
- recommendations,
- clinical scoring, or
- athlete-versus-reference interpretation.

Those functions are outside this prompt unless implemented and validated in a future module.
